# 🎯 YouTube Focus Mode

A powerful Chrome Extension that helps users stay focused on YouTube by removing distractions, filtering content, and tracking watch time.

---

## ✨ Features

### 🚫 Hide YouTube Shorts
- Completely hides Shorts from YouTube.
- Removes Shorts shelf and Shorts videos from multiple pages.

### 📚 Educational Video Filter
- Shows only educational videos.
- Filters entertainment and distracting content.

### ⏩ Auto Skip Intro
- Automatically skips the first few seconds of videos.
- Saves time and improves productivity.

### 🎛️ ON / OFF Toggle
- Instantly enable or disable Focus Mode.
- Preferences are saved using Chrome Storage.

### 📊 Productivity Time Tracker
Tracks:

- Today's watch time
- Weekly watch time

Features:

- Starts only when video is playing
- Stops when:
  - Video is paused
  - Video ends
  - Tab becomes inactive
- Automatically resumes when video plays again

### 🧹 Hide Recommendations
Removes distracting content such as:

- Up Next videos
- Homepage recommendations
- Trending suggestions
- Explore recommendations

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/youtube-focus-mode.git
```

### 2. Open Chrome Extensions

Go to:

```text
chrome://extensions
```

### 3. Enable Developer Mode

Turn ON **Developer Mode** from the top-right corner.

### 4. Load Extension

- Click **Load unpacked**
- Select the `youtube-focus-mode` folder

Done ✅
